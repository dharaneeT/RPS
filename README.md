A simple and interactive Rock-Paper-Scissors web game built using HTML, CSS, and JavaScript. This project demonstrates basic DOM manipulation, event handling, and local storage usage in JavaScript.

🚀 Features
🎮 Play Rock, Paper, Scissors against the computer
🤖 Random computer move generation
📊 Live score tracking (Wins, Losses, Ties)
💾 Score persistence using localStorage
🔄 Reset button to clear scores
🖼️ Visual representation of moves with images
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
📂 Project Structure
├── RPS.html      # Main HTML structure
├── style.css     # Styling and layout
├── script.js     # Game logic and functionality
└── RPSIMAGES/    # Images for moves
⚙️ How It Works
User selects a move (Rock, Paper, or Scissors)
Computer randomly generates a move
Game logic determines the result (Win/Loss/Tie)
Score updates and is stored in browser local storage
UI dynamically updates with results and move comparison
